==Init Machine
$ git config --global user.name "Your Name"
$ git config --global user.email your@email.com
$ git config --global --bool pull.rebase true

$ ssh-keygen -t rsa -C "your@email.com"
$ copy ~/.ssh/id_rsa.pub to the git host (such as github)

To make another machine trust you:
$ ssh-copy-id REMOTE_MACHINE


====Init project
$ cd $(PROJECT)
$ git init
$ git add README.txt
$ git commit -a -m 'init'
$ git remote add origin git@github.com:$(USER)/$(PROJECT).git
$ git push -u origin master


====Clone project with write access
$ git clone git@github.com:AGNUcius/GNUnix.git


to publish:
$ git commit -a -m "some message"; git push


to update:
$ git pull
